Copilot Shop - scaffold

Run locally:

1. Install:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

Notes:
- Uses Vite + React + Tailwind.
- Product data fetched from https://dummyjson.com/
- Orders are saved to localStorage once you "Login" (simple local login).
- Checkout validates with `zod` schema.
